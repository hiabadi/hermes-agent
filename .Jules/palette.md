## 2023-10-24 - Missing Focus Styles on Core UI Button
**Learning:** The core UI Button component (`web/src/components/ui/button.tsx`) was missing keyboard accessibility focus states (`focus-visible`). This meant that many interactive elements across the entire dashboard (using this base component) did not provide visual feedback when navigated via keyboard. Other components like `LanguageSwitcher` and `ThemeSwitcher` had manually implemented focus states, but the reusable generic button component did not.
**Action:** Always verify that foundational UI components (like Button, Input, Select) have robust `focus-visible` states built into their base variants. When building new generic components, ensure they include `focus-visible:outline-none focus-visible:ring-1 focus-visible:ring-ring` (or appropriate thematic equivalents) to guarantee accessibility out-of-the-box for all instances.
